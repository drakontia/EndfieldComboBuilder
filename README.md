# EndfieldComboBuilder

ゲーム「アークナイツ：エンドフィールド」でのチームでのコンボを視覚的に構築するツールです

## 機能

### 基本機能
- **チーム編成**: 最大4人のオペレーターを選択
  - 1番目（自操作キャラ）: 通常攻撃、戦技、連携技、必殺技
  - 2-4番目: 戦技、連携技、必殺技のみ
  - ドラッグ＆ドロップで順番入れ替え可能
- **コンボタイムライン**: 各オペレーターのアクションをタイムライン上に配置
- **必殺技チャージ**: 必殺技ラインに背景グラフでチャージの貯まり具合を表示
- **敵の状態**: タイムライン下部に敵に付いた状態異常・デバフを表示
- **状態異常**: アクションに炎上、凍結、感電、中毒、スタン、脆弱を付与可能
- **コンボ管理**: 
  - コンボの保存（ローカルストレージ）
  - コンボの読み込み
  - コンボのクリア
- **エクスポート機能**:
  - 画像エクスポート
  - URL共有

### 技術スタック
- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **テスト**: 
  - Vitest (ユニットテスト)
  - Playwright (E2Eテスト)
- **デプロイ**: Vercel対応

## 使い方

1. **チーム編成**: 左側のパネルで4つのスロットにオペレーターを選択します
   - 最初のスロット（スロット1）は自操作キャラとなり、通常攻撃が使えます
   - 他のスロットのキャラは通常攻撃がありません（戦技、連携技、必殺技のみ）
   - キャラをドラッグ＆ドロップで並び替えができます
2. **アクション追加**: タイムライン上の各アクションレーンをクリックしてアクションを追加します
3. **アクション編集**: 追加したアクションを右クリックして状態異常を付与できます
4. **アクション削除**: 追加したアクションを左クリックして削除します
5. **必殺技チャージ**: 必殺技のラインには背景にチャージゲージが表示されます
   - 通常攻撃: +5 チャージ
   - 戦技: +15 チャージ
   - 連携技: +20 チャージ
   - 必殺技使用時: チャージリセット
6. **敵の状態**: タイムライン下部に敵に付いた状態異常が表示されます
7. **コンボ保存**: コンボ名を入力して「保存」ボタンをクリックします
8. **コンボ読込**: 「読込」ボタンをクリックして保存したコンボを選択します
9. **画像エクスポート**: 「画像エクスポート」ボタンでタイムラインを画像として保存します
10. **共有**: 「共有」ボタンでURLをクリップボードにコピーします

